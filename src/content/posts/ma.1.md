## **1 如何注册 Telegram？**

现在注册 TG 必须使用 Telegram 手机 APP，不能在 Telegram Web版或 Telegram 电脑版注册；并且注册手机号和 IP 要匹配，比如使用英国手机号码注册时，就必须使用英国的 IP，且要使用干净的 IP（即 IP 欺诈风险接近 0）。

使用 +86 手机号码注册电报，很有可能遇到收不到短信验证码的情况。哪怕可以使用 86 号码注册成功，也很容易遇到双向联系人无法发送私聊，即 +86 号码注册的 TG 只能给 +86 号码注册的用户发送私信，无法给非 +86 号码注册的用户发送私聊，且 86 号码注册的 Telegram 账号比较容易被封。一般比较推荐使用国外的手机号码注册电报。

使用境外手机号码注册 Telegram 的流程：

* 下载 Telegram APP（[苹果版](https://apps.apple.com/app/telegram-messenger/id686449807) ｜ [安卓版](https://play.google.com/store/apps/details?id=org.telegram.messenger)）
* 使用境外手机号码（推荐）或[接码平台](https://hero-sms.com/cn)
* 使用与手机号码相匹配的 IP
* 打开 Telegram APP 输入手机号码和邮箱地址注册
* 输入短信验证码和邮箱验证码
* 设置昵称和头像
* 注册 TG 成功

# **5 设置隐私选项**

TG 上的诈骗非常多，有许多奇怪的人每天拉陌生人进诈骗群，或者发送各种诈骗私聊信息，让人不堪其扰。设置电报的隐私选项非常有必要，尤其是要向陌生人隐藏手机号码。各项设置请参考：

* 手机号码：**不允许任何人**

* 头像：**不允许任何人**

# **6 解除敏感内容限制**

使用 iPhone 手机注册电报后，可能会遇到许多群组无法正常图片或视频内容，需要你在 TG 网页版或 TG 电脑版手动打开解除敏感内容限制的选项，Telegarm 手机版 APP 是没有这个选项的。

在 TG 上解除敏感内容限制的流程如下：

* 打开并登录 Telegram Web 版
* 依次点击左上角 三条杆➡️ Settings
* 点击「Privacy and Security」
* 勾选Sensitive Content下的「Disable filtering」

Tips：使用某些东南亚国家的号码注册的电报，没有该选项，即无法解除敏感内容限制。

# **7 解除双向联系人私聊限制**

使用 +86 号码注册的电报，只能向同样是 +86 手机号码注册的用户发送私信，无法向非 +86 号码注册的用户发送私聊，会提示「抱歉，您现在只能向双向联系人发送消息」。这是因为电报为了防止垃圾信息骚扰或者批量注册机器人的骚扰而对新账号做的限制（尤其是 86 号码）。遇到这种情况，可以手动解除，或升级到 Telegram 会员（可降低被双向限制的概率）。

解除双向联系人私聊限制的方法如下：

* 点击TG官方机器人
* 点击 Start
* 选择底部菜单栏的「But I can’t message non-contacts.」
* 选择「No, I’ll never do any of this!」
* 输入「Accident」或「I don’t know, but I am unable to send any message to anyone.」
